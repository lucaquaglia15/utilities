# Utilities

## rename.sh
- Script to batch rename files with an increasing number (from 1 to n) at the end of the file name
- Put the file plus all the files you want to rename in a separate folder and change the name in line 6
- Run with `./rename.sh`

## convertToken.sh
- Script to convert the myCertificate.p12 file to token usable on the ALICE grid
- Change the path to the myCertificate.p12 file at line 3
- Run with `./convertToken.sh`
